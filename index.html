ADDING SOUNDS

This board automatically lists every audio file in the sounds/ folder of the repository. To add a clip:

  1. Put your .mp3 (or .wav / .ogg / .m4a) file into the sounds/ folder of the repo.
  2. Commit and push (or use GitHub's "Add file → Upload files" in the browser).
  3. Wait ~30–60s for GitHub Pages to rebuild, then reload this page.

The tile label comes from the filename — war_never_changes.mp3 shows as WAR NEVER CHANGES. Use underscores or dashes for spaces.

To remove a sound, delete its file from the sounds/ folder and push.
